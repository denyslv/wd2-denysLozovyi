About This Website

Tools used to build Broken Vinyl Club

I chose a classic HTML, CSS, and JavaScript structure with no framework dependencies. The website uses semantic markup for accessible structure and reusable layout patterns.

Styling is managed with reusable CSS classes, custom properties, and responsive media queries. Interactive behavior is handled with vanilla JavaScript.
